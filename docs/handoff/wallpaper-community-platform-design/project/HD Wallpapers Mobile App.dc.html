React Native implementation notes
NavigationReact Navigation v7 · Stack + Bottom Tab Navigator · Deep linking for wallpaper detail URLs
Image handlingexpo-image-picker for upload · expo-file-system for caching · react-native-fast-image for grid
Download flowreact-native-cameraroll for save to gallery · Progress tracked via upload API events · Background download support
State managementZustand for global state (auth, shortlist, theme) · React Query for API data + caching
ThemeuseColorScheme() for system default · Manual override persisted via AsyncStorage · Same crimson accent in both modes
APIShared REST API with the web platform · JWT auth · Same resolution presets drive both web upload flow and mobile download options
